# 記帳系統  
### 程式介紹  
我們製作的是記帳系統，能夠把日常的支出以及收入等等都記錄或刪除並顯示出來。
### 程式規則  
使用者可以選擇要:  
1.新增記帳  
2.顯示設定  
3.刪除紀錄  
4.查看報表
### 如何安裝執行  
下載完成後，可以使用 Visual Studio 開啟。
